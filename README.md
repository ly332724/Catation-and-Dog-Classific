# 环境  
**Python** 3.1.15  
**Torch** 2.2.2  

# 数据集  
[天池数据集](https://tianchi.aliyun.com/dataset/128500#1)  

# 内容说明  

## 1. 数据集处理  
- 脚本 `data.py` 会将原始数据集分割，并存储到一个新建的小数据集文件夹中，便于后续训练和测试。  

## 2. 模型训练  
- `src` 文件夹中包含不同模型的训练代码，可对多种模型进行调用与训练。  
- 支持修改网络接口以适配需求。  
- 训练完成后，模型会保存到 `model` 文件夹中。  

## 3. 预测  
- 使用 `predict` 脚本进行预测，预测结果会输出到指定路径。  
- 修改脚本中的模型路径为自己保存的路径。
